White_dune ist ein graphischer VRML97/X3D editor, ein simpler NURBS oder Superformula 3D Modeller, ein Animations-Werkzeug und ein VRML97/X3DV Kommandozeilen-Compiler.

Mit white_dune kann man 3D Objekte basteln und animieren, die man auch über WebGL] direkt im webbrowser laufen lassen oder ins RIB renderman Format (für die Film-Erzeugung) andeln kann. Man kann das Ergebnis auch in C, C++ wandeln, so dass ein "normales" OpenGL-Programm entsteht. Neben der normalen Version von white_dune gibt auch ein "-4kids" GUI, das für Anfänger bzw. Kinder geeignet ist.
